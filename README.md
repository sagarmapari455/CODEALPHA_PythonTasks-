CodeAlpha Python Internship – Project Submission

This repository contains the three tasks completed for the CodeAlpha Python Programming Internship.

Completed Tasks:

Task 1: Hangman Game

Task 2: Stock Portfolio Tracker

Task 4: Basic Chatbot



---

TASK 1 – Hangman Game

Description

A simple text-based Hangman game where the player must guess a hidden word.
The player gets 6 incorrect attempts before the game ends.

Features

Random word selection

Tracks correct and incorrect guesses

Prevents repeated guesses

Displays remaining attempts


Concepts Used

random module

loops and conditions

lists

object-oriented programming


How to Run

python hangman.py


---

TASK 2 – Stock Portfolio Tracker

Description

A basic stock tracker where the user enters stock names and quantities.
Stock prices are hardcoded as required by the internship.

Features

Predefined stock price dictionary

Calculates total investment value

Supports multiple stock entries

Option to save results to CSV


Concepts Used

dictionaries

loops

user input

arithmetic

file handling (CSV)


How to Run

python stock_portfolio.py

TASK 4 – Basic Chatbot

Description

A simple rule-based chatbot that responds to basic text messages.

Features

Responds to: hello, hi, how are you, what is your name, bye

Rule-based responses

Runs until the user types "bye"


Concepts Used

functions

if-elif conditions

loops

string processing


How to Run

python chatbot.py
